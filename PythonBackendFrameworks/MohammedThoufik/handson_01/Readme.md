# Hands-On 01 - Web Framework Foundations & Django Project Setup

## Objective

The objective of this hands-on was to understand the fundamentals of web frameworks and set up a basic Django application.

## Tasks Completed

### Task 1 - Web Framework Concepts

Studied:

- Request-Response Cycle
- Middleware Architecture
- WSGI vs ASGI
- MVC vs Django MVT Architecture

### Task 2 - Django Project Setup

Implemented the following:

- Created Django Project: `coursemanager`
- Created Django App: `courses`
- Registered application in `INSTALLED_APPS`
- Configured URL routing using `urls.py`
- Created a simple view function
- Verified application execution using Django development server
